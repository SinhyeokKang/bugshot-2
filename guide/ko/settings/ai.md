# AI LLM 연동

설정의 **AI 모델** 서브탭에서 직접 쓰시는 LLM을 연결할 수 있습니다. BugShot은 별도 AI 서버를 두지 않고, 여러분이 가진 키를 그대로 씁니다(BYOK — Bring Your Own Key). 그러니 내 키, 내 모델로 안심하고 쓰시면 됩니다.

## 연결하기

세 가지만 입력하면 됩니다.

- **Base URL** — LLM API 엔드포인트 주소.
- **API Key** — 해당 서비스에서 발급한 키.
- **Model ID** — 사용할 모델 이름.

OpenAI 호환 엔드포인트를 쓰는 프로바이더라면 대부분 무리 없이 연결됩니다. 연결할 때 해당 도메인 접근 권한을 요청할 수 있는데, 허용해 주시면 됩니다.

![AI 모델 연결](../assets/dummy.jpg)

## 연결하면 켜지는 두 기능

LLM을 연결하면 두 가지 AI 기능이 활성화됩니다. 연결하지 않으면 두 기능 모두 화면에 나타나지 않으니, 필요하실 때 연결하시면 됩니다.

- **AI 스타일링** — 요소를 고를 때 "버튼을 둥글게", "여백을 키워" 같은 자연어 지시로 스타일을 바꿉니다. → [스타일링](../element/styling.md)
- **AI 초안 작성** — 캡처·로그를 근거로 이슈 본문을 자동으로 채웁니다. → [이슈 작성(요소)](../element/issue.md)

> AI도 가끔 실수합니다. 생성된 결과는 제출 전에 한 번만 확인해 주세요.
