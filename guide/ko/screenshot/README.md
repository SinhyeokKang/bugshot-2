# 스크린샷 캡처

화면의 원하는 영역을 캡처하고, 그 위에 주석을 그려 버그 위치를 콕 짚어 이슈로 보내는 모드입니다. 가장 빠르고 직관적이라 처음 쓰기에도 부담이 없습니다.

흐름은 **영역 캡처 → 주석 표시 → 이슈 작성** 순서입니다.

![스크린샷 모드 개요](../assets/dummy.jpg)

## 바로가기

- [스크린샷 캡처](capture.md) — 영역을 드래그해 캡처.
- [어노테이션](annotation.md) — 캡처 이미지에 주석 그리기.
- [이슈 작성](issue.md) — 주석 스크린샷을 담아 제출.
