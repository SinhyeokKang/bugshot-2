# BugShot 개인정보처리방침

**시행일**: 2026년 4월 26일

BugShot(이하 "확장 프로그램")은 사용자의 개인정보를 소중히 여기며, 최소한의 정보만 수집·처리합니다. 본 방침은 확장 프로그램이 어떤 정보를 다루는지 투명하게 안내합니다.

---

## 1. 수집하는 정보

| 정보 | 수집 시점 | 목적 |
|---|---|---|
| Jira 자격 증명 (API Token 또는 OAuth 토큰) | Jira 연동 설정 시 | Jira 이슈 생성 및 첨부 파일 업로드 |
| Jira 사용자 이메일 | Jira 연동 검증 시 (`/myself` API 응답) | 연동 상태 표시 |
| 페이지 DOM 요소 스타일 정보 | 요소 선택 시 | 스타일 비교 및 이슈 본문 생성 |
| 스크린샷·영상 | 캡처/녹화 시 | 이슈 첨부 |

확장 프로그램은 위 항목 외에 사용자의 브라우징 기록, 쿠키, 개인 식별 정보 등을 수집하지 않습니다.

## 2. 정보 저장

모든 데이터는 **사용자의 브라우저 내부**에만 저장됩니다.

- **chrome.storage.local**: Jira 연동 설정, 이슈 기록, 앱 설정
- **chrome.storage.session**: 편집 세션 (탭별, 브라우저 종료 시 자동 삭제)
- **IndexedDB**: 영상 녹화 파일 (로컬 기기 한정)

외부 서버에 사용자 데이터를 저장하지 않습니다.

## 3. 외부 전송

확장 프로그램은 아래 서비스에만 데이터를 전송합니다.

| 대상 | 전송 내용 | 목적 |
|---|---|---|
| Jira REST API (`*.atlassian.net`, `api.atlassian.com`) | 이슈 본문, 스크린샷, 영상 | 이슈 생성 및 첨부 |
| OAuth 프록시 서버 | OAuth 인가 코드 | 토큰 교환 (코드를 access token으로 변환) |

OAuth 프록시 서버는 토큰 교환만 중계하며, 사용자 데이터를 저장하거나 로깅하지 않습니다.

## 4. 제3자 공유

수집한 정보를 제3자에게 판매, 공유, 전달하지 않습니다. 사용자가 직접 Jira에 이슈를 생성하는 경우에만 해당 Jira 워크스페이스로 데이터가 전송됩니다.

## 5. 데이터 삭제

- **확장 프로그램 제거**: `chrome.storage` 데이터가 자동으로 삭제됩니다.
- **영상 데이터**: 브라우저 설정에서 사이트 데이터를 초기화하거나, 확장 프로그램 내 이슈 목록에서 개별 삭제할 수 있습니다.
- **Jira 연동 해제**: 앱 내 Jira 연동 탭에서 연결을 해제하면 저장된 자격 증명이 삭제됩니다.

## 6. 권한 사용 안내

| 권한 | 사용 목적 |
|---|---|
| activeTab | 현재 탭의 DOM 요소 정보 수집 |
| scripting | DOM 선택 및 오버레이 표시를 위한 스크립트 주입 |
| storage | 설정·세션·이슈 기록 저장 |
| tabCapture | 탭 영상 녹화 |
| identity | Atlassian OAuth 로그인 |
